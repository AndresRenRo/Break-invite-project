# Break Invite 🚬☕🍺🍕

**By [The Human Glitch Report](https://thehumanglitchreport.com)**

Send someone an invitation to take a break. Pick your poison — smoke, coffee, beer, pizza, whisky, donuts, gossip, or chaos. Each break has its own vibe, timer, and absurd life advice from Boby, the world's most unhelpful robot.

## How it works

1. **Pick a break type** — 12 options + random roulette
2. **Set the time** — 3, 5, 10, or 15 minutes
3. **Send the link** — share via WhatsApp, DM, carrier pigeon
4. **Your friend opens it** — sees a sealed envelope, taps to open, accepts the invitation
5. **Timer runs** — Boby provides questionable wisdom
6. **Break ends** — epic glitch + aggressive motivational phrase to get back to work
7. **Loop** — "Now invite someone else" keeps the chain going

## Tech

- **React + Vite** — fast, minimal
- **No backend** — invite data is encoded directly in the URL (base64)
- **No database** — zero infrastructure, zero costs
- **Vercel-ready** — deploy in 30 seconds

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import the repo
4. Framework: Vite
5. Deploy

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## Custom domain

Point `break.thehumanglitchreport.com` (or whatever subdomain) to your Vercel deployment.
